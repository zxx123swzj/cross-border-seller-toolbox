# Cross-Border Seller Toolbox

A static website for cross-border ecommerce sellers, with calculators, beginner tutorials, product-fit analysis, shipping estimates, listing helpers, compliance checks, and long-form starter articles.

## Local Preview

```bash
python3 -m http.server 5177
```

Open:

```text
http://127.0.0.1:5177/
```

## Pages

- `index.html` - main toolbox workspace
- `articles.html` - article index
- `article.html?slug=...` - article detail template

## Deploy

This is a plain static site. It can be deployed with GitHub Pages, Vercel, Netlify, Cloudflare Pages, or any static hosting service.
